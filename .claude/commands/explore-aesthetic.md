Generate aesthetic comparison samples for a mini-drama series.

Run:
```
npx tsx src/mini-drama/cli.ts explore-aesthetic -p output/<series>
```

This generates 5-7 style samples showing different visual aesthetics applied to the series' setting. Each sample is saved as a PNG with a comparison HTML page.

Show the user each sample inline. Let them pick one, request hybrids, or ask for more options.

Once chosen, run set-aesthetic with the selected parameters.
