Create a new mini-drama series.

Ask the user for:
1. Series name
2. Concept/premise (what's the story about?)
3. Genre (drama, thriller, romance, etc.)
4. Setting description

Then run:
```
npx tsx src/mini-drama/cli.ts new-series -n "<name>" --concept "<concept>" -g "<genre>" --setting "<setting>"
```

After creation, proceed to aesthetic exploration automatically.
